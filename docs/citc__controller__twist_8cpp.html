Detailed Description

Node file for CITC Controller using velocity commands.

This node requires the following parameters.

Parameters
Gains/K1xValue of \({k_1}_x\)
Gains/K1yValue of \({k_1}_y\)
Gains/K1zValue of \({k_1}_z\)
Gains/K1yawValue of \({k_1}_\psi\)
Gains/K2xValue of \({k_2s}_x\)
Gains/K2yValue of \({k_2}_y\)
Gains/K2zValue of \({k_2}_z\)
Gains/K2yawValue of \({k_2}_\psi\)
Gains/K3xValue of \({k_3}_x\)
Gains/K3yValue of \({k_3}_y\)
Gains/K3zValue of \({k_3}_z\)
Gains/K3yawValue of \({k_3}_\psi\)
Gains/K4xValue of \({k_4}_x\)
Gains/K4yValue of \({k_4}_y\)
Gains/K4zValue of \({k_4}_z\)
Gains/K4yawValue of \({k_4}_\psi\)
Reference_Gains/XGain for \(x\) coordinate used to consider the reference acceleration of the path.
Reference_Gains/YGain for \(y\) coordinate used to consider the reference acceleration of the path.
Reference_Gains/ZGain for \(z\) coordinate used to consider the reference acceleration of the path.
Lambda/XVariable used to limit control actions. It is not recommended to change.
Lambda/YVariable used to limit control actions. It is not recommended to change.
Lambda/ZVariable used to limit control actions. It is not recommended to change.
MassBebop 2 drone mass.
SigmaValue of \( \sigma \)
Disable_CommandsVariable used to run the controller without sending velocity commands. This can be useful to check if the safe zone is working properly.
Topics/Command_TrajectoryTopic used to send and receive the command trajectory between nodes.
Topics/PoseTopic used to receive the drone position data.
Topics/CMD_VelTopic used to send the velocity commands to the drone.
Topics/TafeOffTopic used to send the takeoff command to the drone.
Topics/LandTopic used to send the land command to the drone.
Topics/CSV_BeginTopic used to communicate when the trajectory begins and the data_to_csv node should start saving the data.
Topics/CSV_EndTopic used to communicate when the trajectory begins and the data_to_csv node should stop saving the data.
Normalize/Max_Horizontal_SpeedMaximum horizontal speed. Used to normalize the speed in the \(x\) and \(y\) coordinates.
Normalize/Max_Vertical_SpeedMaximum vertical speed. Used to normalize the control action in the \(z\) coordinate
Normalize/Max_Rotation_SpeedMaximum rotation speed. Used to normalize the yaw angle control action.
Safe_Zone/XMaximum position value allowed in the \(x\) coordinate. If the drone reaches a position that exceeds this value, the drone will land.
Safe_Zone/YMaximum position value allowed in the \(y\) coordinate. If the drone reaches a position that exceeds this value, the drone will land.
Safe_Zone/ZMaximum position value allowed in the \(z\) coordinate. If the drone reaches a position that exceeds this value, the drone will land.
Max_Speed/XMaximum velocity allowed in \(x\) coordinate.
Max_Speed/YMaximum velocity allowed in \(x\) coordinate.
Max_Speed/ZMaximum velocity allowed in \(x\) coordinate.
Max_Speed/YawMaximum angular velocity allowed for yaw angle.

It is recommended to pass the parameters using the following YAML files.

  • citc_controller_twist.yaml
  • topics.yaml
  • normalize_twist.yaml
  • safe_zone.yaml
  • max_speed.yaml
