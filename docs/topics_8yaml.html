Detailed Description

YAML file to set the topics used by the nodes.

The following parameters can be configured in this file.

Parameters
Topics/Command_TrajectoryTopic used to send and receive the command trajectory between nodes.
Topics/PoseTopic used to receive the drone position data.
Topics/CMD_VelTopic used to send the velocity commands to the drone.
Topics/TafeOffTopic used to send the takeoff command to the drone.
Topics/LandTopic used to send the land command to the drone.
Topics/CSV_BeginTopic used to communicate when the trajectory begins and the data_to_csv node should start saving the data.
Topics/CSV_EndTopic used to communicate when the trajectory begins and the data_to_csv node should stop saving the data.
